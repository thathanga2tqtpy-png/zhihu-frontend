# Lộ trình phát triển

1. **Giai đoạn 1: Khởi tạo**
   - Thiết lập Next.js project.
   - Cấu hình file CSS và base styles.
   - Tạo cấu trúc thư mục.

2. **Giai đoạn 2: Phát triển Core (Đọc truyện)**
   - Xây dựng trang danh sách truyện.
   - Xây dựng trang chi tiết truyện (Reading View).
   - Fetch dữ liệu từ DB (mock data trước nếu cần).

3. **Giai đoạn 3: Tính năng tương tác**
   - Đánh giá (Ratings).
   - Bình luận (Comments).
   - Theo dõi truyện (Follows).

4. **Giai đoạn 4: Hoàn thiện**
   - Tối ưu SEO và hiệu năng.
   - Kiểm tra trải nghiệm trên di động (Responsive).
